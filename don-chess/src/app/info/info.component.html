This is a classic (western style) chess-game platform, where you can play chess with your friends.
There are two built-in chess-bot, with very low A.I. (they move rather random)
Used technologies:
Backend:
- Java 11
- Java Spring Boot (Core, Data, Security, MVC)
- PostrgreSQL
- Hibernate/JPA & QueryDSL
- Tomcat
- JUnit
- Jackson
